# S4. 브리프 — 파트별 포맷

파트 분류가 브리프 형태를 자동 결정한다.

| 파트 | 브리프 포맷 | 구성 |
|------|-----------|------|
| 기획 | PRD초안 | 배경→의사결정→시나리오→레퍼런스→기획자질문 |
| 디자인 | 디자인브리프 | 목적→사용자맥락→레퍼런스→제약→디자이너질문 |
| 개발 | 기술스펙 | 목적→요구사항→의존성→인프라→개발자질문 |
| 비즈니스 | 액션브리프 | 목적→방향→성공조건→담당·일정→참고 |

**헤더:** `액션:[접두어-#] | 등급:[S/A/B] | 마일스톤:M[#] | 파트:[기획/디자인/개발]`

**등급별 분량:**
- S등급 = 풀포맷
- A등급 = 핵심 3섹션
- B등급 = 1페이저
