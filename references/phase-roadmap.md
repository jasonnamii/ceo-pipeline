# S2. 로드맵 — 중간 지점(마일스톤)

| 항목 | 내용 |
|------|------|
| 성격 | 청사진에 도달하기 위한 경유지. "이 시점에 우리는 이런 상태" |
| 입력 | 청사진 + 현재 상태 + 외부 변수 |
| 갱신 | 상시. 마일스톤 달성/변경 시 업데이트 |

**포맷:** 테이블(#, 마일스톤, 시점, 상태, 달성조건). 마일스톤은 "상태"로 기술 (≠할일). 과제 테이블 금지(→액션리스트). 순서/의존 명시.

**캐스케이드 연결:** 각 마일스톤의 "달성조건"이 갭 탐지의 비교 기준이 된다. 달성조건이 측정 가능해야 갭이 명확하게 도출되기 때문이다.
